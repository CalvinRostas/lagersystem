# **Requirement 8: Display Item Details**

**Title:** Display Item Information
**Number:** SRS-008
**State:** approved
**Priority:** high

# Header

**When** an item is selected via search or list selection,
**the system shall** display the item, item qr code and the assigned storage location in a structured view.

# Acceptance Criteria:

- **Pre-condition:**
  - The item exists in the system and has an assigned storage location.
- **Content:**
  - The item qr code is displayed clearly and unambiguously.
  - The item information are displayed clearly.
  - The assigned storage location is displayed in a human-readable format.
- **Usability:**
  - Text is readable without zooming on standard smartphone screens.
- **Consistency:**
  - The displayed information matches the stored item-location mapping.
- **Verifiability:**
  - When clicked on an item, the displayed data matches the local storage item entry.
